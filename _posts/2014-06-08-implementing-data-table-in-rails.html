\n\nData Table help us to turn a plain HTML table into one with pagination, sorting and searching and all through JavaScript. It has better user interface and provides many options to the user.\r\n\r\nYou  can learn more about data tables from its official website http://datatables.net . Here is a simple example of integrating data table in your rails app.\r\n\r\nStep 1\r\n\r\nAdd the gem in your gem file\r\n
gem 'jquery-datatables-rails', git: 'git://github.com/rweng/jquery-datatables-rails.git'
\r\nRun the bundle install.\r\n\r\nStep 2\r\n\r\nWe can now set DataTables up with the asset pipeline.\r\n\r\nIn your application.js add the following line\r\n
//= require dataTables/jquery.dataTables
\r\nIn your application.css add the following line\r\n
*= require dataTables/jquery.dataTables
\r\nStep 3\r\n\r\nNow we have installed the data table in the application.In order to display the data table effect in your listing page ,modify it like below\r\n
\r\n
\r\n
<table
    id=\"product_lists\">\r\n<thead>\r\n  <tr>\r\n    <th>Product
    Name</th>\r\n    <th>Category</th>\r\n    <th>Price</th>\r\n 
    </tr>\r\n</thead>\r\n<tbody>\r\n  <% @products.each do |product|
    %>\r\n    <tr>\r\n      <td><%= link_to(product.name, product)
    %></td>\r\n      <td><%= product.category %></td>\r\n     
    <td><%= product.price %></td>\r\n    </tr>\r\n  <%
    end %>\r\n</tbody>\r\n</table>
\r\n
\r\n
\r\nStep 4\r\n\r\nNow go to the coffee script file product.js.coffee and add the following block.\r\n
\r\njQuery ->\r\n
    \ $(\"#product_lists\").dataTable()
\r\nYou can set more config options from data tables offical site http://datatables.net like sorting specific columns,disable sorting from columns and may more.\r\nWhen you reload the page,you can see your listing page in better UI with sorting,searching and pagination functionality.
